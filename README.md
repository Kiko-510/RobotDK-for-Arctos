# RobotDK-for-Arctos
1. follow the steps: https://github.com/Arctos-Robotics/RoboDK
If you only want to test the gcode of the program, there is no need to download RobotDK. If you want to connect the Arctos robot arm to the computer, you must download UGS.

2. Open the arctos_demo.rdk in RobotDK, everyone can try to open the model I have built and write simple movements for the Arctos robotic arm. (Please remember not to modify the tool of the robotic arm!!! This is the TCP position that I had previously set up according to the 3D printing nozzle of the robotic arm.)
